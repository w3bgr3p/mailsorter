===== Ман =====

1. Настройка:
- Убедитесь, что у вас установлен Python 3.10 или новее.
- Установите необходимые библиотеки с помощью команды: pip install -r requirements.txt
- Запустите бота с помощью команды: python bot.py

2. Команды:

/start - Запуск бота и вывод списка доступных команд с коротким пояснением.

/add_keyword <ключ> <значение1> <значение2> ... - Добавление ключевых слов в первый словарь.
Пример: /add_keyword фрукты яблоко банан

/add_secondary_keyword <ключ> <значение1> <значение2> ... - Добавление ключевых слов во второй словарь.
Пример: /add_secondary_keyword цвет красный зеленый

/help - Вывод этого мануала.

3. Работа бота:

В первый словарь добавляем ключевые слова (или отправителей например) по которым хотим получать уведомления
Во второй словарь добавляем алиасы ваших мультов.
Трик (псевдонимы с решетками вначеле позволят использовать функционал тегов)
